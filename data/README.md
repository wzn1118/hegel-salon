# Runtime Data

This directory is intentionally kept empty in the public GitHub source release.

At runtime, Hegel Salon may create local state here, including:

- `data/logs/`
- `data/uploads/`
- `data/auth/`
- `data/computer-browser-sessions/`
- `data/corpus/generated/`

Do not commit generated runtime data, browser profiles, user databases, sessions, or uploaded files.

The only corpus material intended for public GitHub distribution is
`data/corpus/texts`, which contains German and English public-domain or openly
mirrored primary text files.
