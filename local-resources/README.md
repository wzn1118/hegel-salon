# Local Resources

This public repository does not bundle copyrighted Hegel texts, private PDFs, or local OCR materials.

If you want to use local corpus features, place your own materials in this directory after cloning the repo.
Public deployments should only mount this directory when the operator has redistribution or processing rights
for the materials being mounted.

Recommended pattern:

- keep your source texts out of version control
- add only files you have the right to store and process
- do not commit personal documents or licensed corpora
- for private servers, mount this folder as a persistent volume instead of committing it
